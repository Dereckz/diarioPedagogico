Bien!!, ahora se mostrara una serie de opciones, deberas seleccionar la opción "Colaboración".

Ahora bien, ya podras crear tús colaboraciones o grupos.

Para ello bastara con pulsar el boton

Ahora bien, escribe el nombre con el cual se identificara tu grupo a como se muestra en la siguiente imagen

Genial!! Si te das cuenta ya se mostraran tus grupos en el apartado de Colaboraciones.

Y asi podras crear las colaboraciones que requieras.

Agregar Amigos

Bien!! Ahora debemos agragar colaboradores al grupo, para eso debemos seleccionar un grupo

Para agregar participantes a una colaboración bastara con Escribir algunas letras del nombre de la otra parsona en el buscador, tal y como se muestra a en la siguinete imagen

Una vez que encontremos a la persona deseada, mantenemos pulsado el nombre y enseguida se mostrara el mensaje para agregarlo al grupo.

Finalmente ya podemos Proceder a realizar publicaciones.

¡¡Genial todo esta listo!!

Ahora ya podras realizar publicaciones en cualquiera de los grupos que hayas creado. Mi Diario

Publicar

Para realizar una publicación desde el apartado principal Mi Diario seleccionar el boton


Posteriormante se mostrara el apartado donde se se podra redactar el contenido de la publicación, en cada publicacion sera necesario rellenar el primer campo, los otros campos se podran rellenar en otro momento de acuerdo al siguiente apartado Editar Publicación.

De forma conjunta podemos adjuntar archivos como son ímagenes, fotos, documentos PDF, y grabaciones de audio.


Finalmente para registrar la publicación solo se debe presionar el boton de Enviar, ubicado en la parte superior derecha de la pantalla





Finalmete La publicación se vera reflejada, en la pantalla principal de publicaciones como se muestra a continuación.
